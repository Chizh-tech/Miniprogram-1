# 潜水日志小程序

一个面向潜水记录场景的微信小程序，支持日历管理、状态标记、图文视频记录、位置与天气自动采集。

## 当前版本功能

- 📅 日历与列表双视图：可在首页切换“潜水日历 / 潜水日志”。
- 🏷️ 潜水状态标记：支持“想去潜水 / 出发潜水 / 正在潜水 / 潜水归来”。
- 🧭 今天特别标识：日历中的今天显示“今”角标。
- ⏱️ 未来日期限制：未来日期可点选，但不能新建日志。
- ✏️ 日志编辑：支持标题、正文、状态、位置、天气、图片、视频。
- 📷 图片管理：最多 9 张图片。
- 🎬 视频管理：最多 3 段视频（单段最长 60 秒）。
- 📍 自动定位：可获取当前位置并反向解析地址。
- 🌤️ 自动天气：根据定位坐标获取实时天气。
- 🗂️ 日志详情：查看完整内容、媒体资源、状态标签，并支持编辑/删除。

## 页面说明

- 首页：月历展示、状态图例、当月日志列表、全部日志列表。
- 编辑页：新建/编辑日志、选择潜水状态、上传媒体、定位与天气调试信息。
- 详情页：展示日志详情与潜水状态，支持编辑和删除。

## 技术与存储

- 框架：微信小程序原生。
- 数据存储策略：优先云数据库，未启用云能力时回退到本地存储。
- 媒体存储策略：
  - 启用云开发时：图片/视频上传到云存储。
  - 未启用云开发时：文件保存在小程序本地用户目录。

## 快速开始

1. 使用微信开发者工具打开项目根目录。
2. 检查并配置 `config.js`。
3. 在小程序管理后台配置合法域名。
4. 重新编译并预览。

## 配置说明

`config.js` 关键字段：

```js
const config = {
  CLOUD_ENV_ID: '你的云开发环境ID',
  DIARY_COLLECTION: 'diaries',
  WEATHER_API_KEY: '你的 OpenWeatherMap Key',
  TENCENT_MAP_KEY: '你的腾讯地图 Key',
  WEATHER_API_URL: 'https://api.openweathermap.org/data/2.5/weather',
  GEOCODING_API_URL: 'https://apis.map.qq.com/ws/geocoder/v1/'
};
```

小程序后台需要添加以下服务器域名：

- `https://api.openweathermap.org`
- `https://apis.map.qq.com`

说明：

- 若未配置天气/地图 Key，日志核心功能仍可使用。
- 未配置时，地址可能降级为经纬度，天气可能显示获取失败。

## 云开发初始化（可选）

1. 在微信开发者工具开通云开发环境。
2. 将 `CLOUD_ENV_ID` 设置为你的环境 ID。
3. 创建云数据库集合（默认 `diaries`）。
4. 重新编译，应用会自动优先使用云端存储。

## 常见问题

### 1. 预览时报语法错误

- 优先查看报错里的真实文件路径和行号。
- 如果路径是 `vscode-chat-code-block://`，通常是临时编辑器内容，不是项目源码。

### 2. 天气或地址获取失败

- 检查 `WEATHER_API_KEY` / `TENCENT_MAP_KEY` 是否有效。
- 检查小程序后台是否正确配置服务器域名。
- 检查定位权限是否已授权。

### 3. 云存储未生效

- 检查 `CLOUD_ENV_ID` 是否正确。
- 检查当前账号是否已开通并可访问对应云环境。
